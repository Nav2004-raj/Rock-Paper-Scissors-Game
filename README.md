# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and test your luck!

🚀 Features
🎮 Play Rock, Paper, Scissors against the computer
⚡ Instant result display (Win / Lose / Draw)
📊 Score tracking system
🎨 Clean and responsive UI
🔄 Replay as many times as you want

🛠️ Tech Stack
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and interactivity

📂 Project Structure
rock-paper-scissors/
│
├── index.html      # Main HTML file
├── style.css       # Styling file
├── script.js       # Game logic
└── README.md       # Project documentation

🎯 How to Play
Choose one option:
Rock ✊
Paper ✋
Scissors ✌️
The computer will randomly select an option.

The winner is decided based on:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
The result and score will update instantly.

🧠 Game Logic
The computer choice is generated using Math.random()
User and computer choices are compared using conditional statements
Score updates dynamically after each round
